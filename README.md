# Network-and-Security-Project
A Secure Small Enterprise Network (cisco packet tracer)
# project overview
This project focuses on building and testing secure network infrastructures while applying modern cybersecurity principles. It is designed to demonstrate practical implementations of network design, monitoring, and defense mechanisms against common threats.
